# 🎵 Spotify Data Analytics

## 📌 Project Overview

This project analyzes Spotify music streaming data using Python, SQL, Excel, and Power BI to uncover trends in song popularity, artist performance, genres, audio features, and user listening behavior. The project includes data cleaning, exploratory data analysis (EDA), interactive visualizations, and business insights.

---

## 🎯 Objectives

- Clean and preprocess the dataset
- Perform Exploratory Data Analysis (EDA)
- Analyze song popularity and artist performance
- Explore genre and audio feature trends
- Build an interactive Power BI dashboard
- Generate actionable business insights

---

## 📊 Key Analysis

- Top Artists by Popularity
- Most Streamed Tracks
- Genre-wise Song Distribution
- Audio Feature Analysis (Danceability, Energy, Valence, Tempo)
- Explicit vs Non-Explicit Songs
- Popularity Distribution
- Release Year Trends

## 🚀 Business Insights

- Identified the most popular artists and tracks.
- Analyzed genre distribution and listener preferences.
- Explored relationships between audio features and song popularity.
- Examined music release trends over time.
- Built an interactive dashboard for data-driven music insights.
